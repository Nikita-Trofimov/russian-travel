# Путешествия по России
Лендинг о путешествии от столицы до Байкала на электричках  
**Использовано:**
1. Grid Layout
2. Медиа запросы
3. Flex-box
4. Абсолютное позиционирование
5. Анимации с помощью CSS
6. Структура построена по БЭМ методологии
